pptx-workflows:
commenet: Detailed middle-focused procedures for PPTX engineering
workflow:
goal: Read presentation content without altering it.
steps:
step: Convert deck text to markdown using markitdown for quick triage.
step: Unzip with skills/pptx/ooxml/scripts/unpack.py into a scratch directory.
step: Inspect ppt/presentation.xml for slide ordering and metadata.
step: Check ppt/slides/slideN.xml plus notesSlides, comments, layouts, masters, and theme files for styling intel.
step: Use grep/rg to search for color nodes (<a:solidFill>, <a:srgbClr>) or font nodes (<a:fontScheme>).

workflow:
goal: Modify an existing deck at the XML layer.
steps:
step: Read ooxml.md completely.
step: Unpack: python skills/pptx/ooxml/scripts/unpack.py deck.pptx work/unpacked.
step: Edit ppt/slides/slideN.xml, layouts, masters, theme, etc., as needed.
step: Validate after each logical chunk: python skills/pptx/ooxml/scripts/validate.py work/unpacked --original deck.pptx.
step: Repack when clean: python skills/pptx/ooxml/scripts/pack.py work/unpacked deck-new.pptx.
workflow:
goal: Build a new deck from a template while preserving design intent.
steps:
step: Extract template text via python -m markitdown template.pptx > template-content.md and read fully.
step: Generate thumbnail grids: python scripts/thumbnail.py template.pptx analysis --cols 4.
step: Author template-inventory.md enumerating every slide (index, layout, purpose).
step: Write outline.md mapping your new content sections to template slide indices (0-based).
step: Use scripts/rearrange.py template.pptx working.pptx idx0,idx1,... to duplicate/reorder slides.
step: Extract text inventory: python scripts/inventory.py working.pptx text-inventory.json and study the full JSON (no slicing).
step: Prepare replacement-text.json referencing actual slide/shape ids, copying necessary paragraph properties (bullets, alignment, fonts, colors, spacing).
step: Apply replacements: python scripts/replace.py working.pptx replacement-text.json output.pptx; fix any validation overflow errors reported.
